Bring your contacts in?

Your Drive is empty, so it looks like you're just getting started. You can import your existing contacts as a .vcf (vCard) file — here's how to export one:

📇 Google Contacts (any device)

Go to contacts.google.com → select the contacts you want (or "Select all") → click Export → choose vCard (for iOS Contacts) format → download the .vcf file.

🤖 Android

Open the Contacts app → tap the menu (☰) → Settings → Export → choose "Export to .vcf file" → save it, then share/move it to a device where you can upload it here. (Exact wording varies a bit by phone brand.)

🍎 iPhone / iPad

Easiest path is via iCloud: on a computer, go to icloud.com/contacts → select all contacts (⌘/Ctrl+A) → click the gear icon in the bottom-left → Export vCard.

Once you have the .vcf file, use the button below (or the Sync tab any time later) to bring it in.
